# Action402 Brand Package

This folder is the standalone brand package for Action402.

## Contents

- `Branding-Guide.md` - full brand guide: positioning, logo rules, colors, typography, voice, UI style, merch.
- `Slogans.md` - slogan and tagline options in English and Russian.
- `Landing-Page-Brief-for-Subagent.md` - implementation brief for a subagent building the landing page.
- `brand-tokens.json` - machine-readable colors, typography, radii, and usage tokens.
- `assets/logo-action402.svg` - logo asset for use in pages.
- `assets/logo-action402-source.svg` - explicit source SVG for logo editing/export.
- `assets/brand-board-preview.png` - generated visual brand board preview.
- `reference/brand.html` and `reference/brand.css` - visual brand guide reference page.

## Primary Direction

Use the `A4 Relay` logo direction and keep the core positioning:

```text
Paid execution infrastructure for autonomous agents.
```

Core slogan:

```text
Pay. Execute. Prove.
```
